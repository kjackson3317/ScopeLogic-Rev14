# ScopeLogic Official Logo Standard

**Status:** Official
**Effective revision:** ScopeLogic Revision 14.5
**Company:** ScopeLogic LLC
**Tagline:** Identify. Clarify. Rectify.

## Official source artwork

The file below is the controlling logo artwork moving forward:

- `public/brand/scopelogic-official-logo-source.png`

This artwork is the approved ScopeLogic logo and combines:

- Structured cabling / RJ45 representation
- CCTV camera representation
- Access-control reader / lock representation
- Audio-visual microphone and headset representation
- ScopeLogic `S` symbol
- ScopeLogic wordmark
- `Identify. Clarify. Rectify.` tagline

## Approved application assets

The source artwork is also provided in approved application crops:

- `scopelogic-logo-full.png` - full logo for PDF covers, release packages, proposals, and prominent client-facing locations.
- `scopelogic-logo-mark.png` - symbol for the application sidebar, icons, compact document headers, and small placements.
- `scopelogic-wordmark.png` - horizontal wordmark for application and PDF headers.
- `scopelogic-tagline.png` - standalone tagline artwork.
- `scopelogic-app-icon.png` - application and browser icon.

## Color and modification rules

The official palette is OD green, black, and white. Do not redraw, re-typeset, stretch, recolor, rotate, rearrange, or substitute any device symbol without an approved logo revision. Cropping the approved source into the listed full-logo, mark, wordmark, tagline, and application-icon variants is permitted.
