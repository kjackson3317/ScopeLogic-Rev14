# ScopeLogic Revision 14.6

## Sidebar branding

The ScopeLogic wordmark is now rendered in its original black/OD-green artwork on the white sidebar name panel. The prior CSS filter converted the complete image to white, which made the name appear blank.

## Project Library calendar

The Project Library now contains an Important Dates calendar. Users can:

- Move between months.
- Select a date.
- Add a categorized milestone.
- Assign the milestone to a project.
- Review and remove dates from the selected day.

Calendar entries are stored in the same browser workspace as projects and remain available after reload.

## Dashboard current drawings

The Dashboard now lists every Project Document that meets both conditions:

- Document Type = Drawings
- Current Document = checked

The list displays the saved display name, original filename, revision, and issue date.

## Dashboard contract details

Each project now stores editable contract details:

- Product offering
- Engagement basis
- Pricing tier
- Contract/proposal number
- Contract amount
- Contract status
- Start date
- Target completion
- Contract notes

Pricing guidance follows the established ScopeLogic pricing structure:

- Product 1: $3,000–$10,000 standalone.
- Product 2: Small/Medium/Large standalone and Product 1 add-on pricing.
- Product 3: $4,000–$9,000 standalone or $2,500–$6,000 add-on.
- Product 4: Light, Standard, and Advanced monthly-retainer ranges, with the established onboarding range when no prior ScopeLogic service exists.

The dashboard also displays the established Products 1–3 review-meeting allowance and travel policy.
