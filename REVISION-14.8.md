# ScopeLogic Revision 14.8

## Workflow corrections

- Global top-toolbar New SLR action removed.
- Contractor checklist inclusion is controlled by the Contractor Checklist Scope Item text field.
- Checklist PDFs use that field instead of the Internal Matrix Scope Item.
- Blank checklist text omits the SLR from the checklist.
- Newly submitted SLR entry fields reset for the next record.
- Successful email sends close the composer before confirmation.

## Administration corrections

- Email Settings displays server configuration status.
- Invalid Resend API-key errors are translated into actionable deployment instructions.
- ScopeLogic Standards now documents application behavior and deliverable mapping.
